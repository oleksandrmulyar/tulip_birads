# tulip_birads

Статичний фронтенд-конструктор BI-RADS MRI протоколу з повністю окремою логікою для:
- Right breast;
- Left breast.

## Що вміє
- додавати до **20 утворень на кожну залозу**;
- для кожного утвору вказувати характеристики за MRI-лексиконом:
  - тип: focus / mass / non-mass enhancement;
  - для mass: форма, контури, внутрішнє підсилення;
  - для non-mass enhancement: розподіл, внутрішній патерн;
  - розмір у форматі **Розмір (X×Y мм)** з двома компактними полями;
  - кінетика: initial + delayed curve;
  - DWI;
  - окремий BI-RADS для кожного утвору;
- задавати локалізацію (циферблат, відстань від соска, глибина);
- автоматично ставити маркери утворів на схемах (коронарна + сагітальна);
- мати **окремі опитувальники** для правої/лівої залози;
- автоматично обчислювати **сумарний BI-RADS** для кожної залози як максимальний серед утворів (з посиланням на утвір, що визначив категорію);
- генерувати готовий текст протоколу з розділенням на Right/Left та автопідсумком BI-RADS по кожній залозі.

## Запуск

